## FAQ

**Q: Do I need a Razorpay account?**
A: For UPI/GPay payments, I can set it up with a simple UPI QR code. For cards/netbanking, you'll need a Razorpay account (free to create).

**Q: Can you use my own domain name?**
A: Yes! The Premium package includes domain setup. I can also help you buy one.

**Q: Will this work on mobile phones?**
A: Absolutely. Every site I build is fully responsive and works on all devices.

**Q: Can I add more products later?**
A: Yes. The Standard and Premium packages include an admin panel where you can add/edit products yourself.

**Q: How is this different from Shopify?**
A: No monthly fees, fully custom code, and native UPI/GPay support without payment gateways charging extra.
